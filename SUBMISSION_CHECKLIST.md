# Submission Checklist

Use this checklist before sending the application.

## Required Artifacts

- GitHub repository link
- Public `predictions.csv` link
- `results.json` generated by the official `solution.py`
- `predictions.csv` generated by the official `solution.py`
- `SOLUTION.md` report

## Final Run

Run in Google Colab or Kaggle with GPU:

```bash
git clone https://github.com/mart1ny/summer-school-application-china.git
cd summer-school-application-china
pip install -r requirements.txt
python solution.py
```

Expected output files:

- `results.json`
- `predictions.csv`

## Sanity Checks

- `results.json` contains `feature_dim = 896`.
- `results.json` contains `avg_test_accuracy` close to `0.7500`.
- `predictions.csv` has exactly two columns: `id`, `label`.
- `predictions.csv` has 100 rows plus the header.
- Current final predictions distribution: 31 labels `0`, 69 labels `1`.
- Labels are only `0` and `1`.
- `solution.py`, `model.py`, and `evaluate.py` are unchanged.
- No local machine paths appear in the code.
- No labels from `data/test.csv` are used.
